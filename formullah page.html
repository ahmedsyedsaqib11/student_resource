Chapter 5: Square and Square Roots

Formulas

  • Product Rule: √(a * b) = √a * √b. Example: √(4 * 9) = √4 * √9 = 2 * 3 = 6.
  • Quotient Rule: √(a / b) = √a / √b. Example: √(16 / 4) = √16 / √4 = 4 / 2 = 2.
  • Power Rule: (√a)^n = √(a^n) Example: (√4)^2 = √(4^2) = √16 = 4.
  • Simplifying Expressions: √(a^2) = a. ...
  • Cube of a Number: (a)^3 = a * a * a. ...
  • Cube Root of a Number: ∛(a^3) = a.
